 [ 1 はじめに ================================================================ ]

    これはMikanライブラリと、それをすぐに使えるプロジェクトを作るウィザードをイ
  ンストールするインストーラーです(ライブラリのみも可能)。

    MikanライブラリはWindows向けのDirectXを使用したゲームを作るフレームワークラ
  イブラリで、使用には DirectX SDK が必要です。

 [ 2 インストール ============================================================ ]

    まず始めに、DirectX SDK と Visual Studio もしくは Visual C++ Express をイン
  ストールしてください。

    その後、MikanInstaller2010.exe を実行すると、インストールが可能です。

    その際にWindows7を使用している人は、右クリック->互換性->互換モードでこのプロ
  グラムを実行する->Windows vista を選択してください。

 [ 3 アンインストール ======================================================== ]

    アンインストールは2箇所のファイルをすべて削除するだけです。

    ライブラリはインストールした場所(初期値 C:\lib\Mikan)のファイルをすべて削除
  してください。

    ウィザードのファイルも同時に削除できますが、ウィザードの登録は削除しきれない
  ので、マイドキュメント(C:\Users\ユーザー名\Documents)の中にある
  Visual Studio 2012フォルダ内の Wizards を削除してください。

    レジストリに関しては、
HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Uninstall\MikanLibrary
  のキーを削除すれば良いです。

    Mikanライブラリのインストール場所に生成される、uninstall.batを実行することで
  これらの作業を行えます。

 [ 4 その他 ================================================================== ]

    インストーラーの作業はZIPファイルの解凍及びコピー、ウィザードのファイルを一
  部書き換えてコピーする程度です。

    data フォルダ内にライブラリのファイルがあるので、最悪の場合手動でインストー
  ルすることも可能です。
